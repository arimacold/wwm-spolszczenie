# Spolszczenie Where Winds Meet

Polskie tłumaczenie gry **Where Winds Meet** wraz z automatycznym instalatorem.

## Pobieranie
Przejdź do zakładki **Releases** i pobierz najnowszą wersję instalatora.

## Funkcje
- automatyczna instalacja
- wykrywanie Steam
- backup oryginalnych plików
- przywracanie starego tłumaczenia

## Wymagania
- Windows 10 / 11
- Wersja Steam gry
